## Local Development

Run `elm-reactor` and visit `http://localhost:8000/examples/name-face-game.html` for sample people 

## Building

Build chrome extension:

``` 
elm-make examples/name-face-game.elm --output chrome-extension/src/inject/elm-game.js
```
